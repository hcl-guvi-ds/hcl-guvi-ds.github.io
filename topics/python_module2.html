Lesson 1

What is a Variable?

A variable is a named container that stores a value in your program's memory. Think of it like a labelled box — the label is the variable name, the contents are the value.

name = "Alice"
name "Alice" str
age = 25
age 25 int
height = 5.6
height 5.6 float
active = True
active True bool

Hover over a box to see it light up! Each variable stores a different type of data.

How Assignment Works

score = 95
→
memory location "score" 95
The = sign is the assignment operator. It stores the right-hand value into the left-hand variable name.
variables.py
# Create variables
name   = "Alice"
age    = 25
height = 5.6
active = True

# Print them
print(name)    # Alice
print(age)     # 25
print(height)  # 5.6
print(active)  # True

# Reassign — variables can change!
age = 26
print(age)     # 26
Variables are case-sensitive! name, Name, and NAME are three completely different variables.

Valid Variable Names

✅ my_name
❌ 2name — can't start with digit
✅ _private
❌ my name — no spaces allowed
✅ score2024
❌ class — reserved keyword
✅ firstName
❌ my-var — hyphens not allowed
Lesson 2

The 4 Basic Data Types

Every value in Python has a type. The type tells Python what kind of data it is and what operations you can do with it. Click each tab to explore.

🔢

Integer int

Whole numbers — positive, negative, or zero. No decimal point. Used for counting, indexing, and exact math.

42 -7 0 1_000_000
integers.py
x = 42
y = -10
big = 1_000_000   # underscores for readability

print(x + y)          # 32
print(x * y)          # -420
print(big)            # 1000000
print(type(x))        # <class 'int'>
Python integers have unlimited precision — you can store numbers as large as your RAM allows!
💧

Float float

Numbers with a decimal point. Used for measurements, scientific values, and anything requiring precision.

3.14 -0.5 2.0 1.5e10
floats.py
pi = 3.14159
temp = -0.5
sci  = 1.5e10     # scientific notation: 15000000000.0

print(pi * 2)        # 6.28318
print(round(pi, 2)) # 3.14
print(type(pi))      # <class 'float'>
Be careful with float precision! 0.1 + 0.2 gives 0.30000000000000004 in Python — a quirk of how computers store decimals.
📝

String str

Text data — any sequence of characters wrapped in single or double quotes. Strings are immutable (can't be changed in-place).

"Hello" 'World' "42" ""
strings.py
greeting = "Hello, Python!"

print(len(greeting))          # 15 (length)
print(greeting.upper())       # HELLO, PYTHON!
print(greeting.lower())       # hello, python!
print(greeting[0])            # H (indexing)
print(greeting[0:5])          # Hello (slicing)
print("Python" in greeting)   # True
Strings have dozens of built-in methods! Try .strip(), .replace(), .split(), and .count().
✅

Boolean bool

Only two possible values: True or False. The foundation of all decision-making in Python.

True False 5 > 3 x == y
booleans.py
is_raining = False
is_sunny   = True

print(is_raining)           # False
print(is_sunny)             # True
print(5 > 3)               # True
print(10 == 20)            # False
print(not is_raining)       # True
print(is_sunny and not is_raining)  # True
In Python, True equals 1 and False equals 0 when used in math. Try print(True + True) — it outputs 2!

Lesson 3

Type Conversion

Python lets you convert values from one type to another using built-in functions. This is called casting or type conversion.

Conversion Functions

str"42"
→ int() →
int42
→ float() →
float42.0
→ str() →
str"42.0"
conversion.py
# str → int
s = "100"
n = int(s)
print(n + 5)        # 105  (not "1005"!)

# int → float
x = float(7)
print(x)            # 7.0

# number → str (for joining text)
age = 25
msg = "I am " + str(age) + " years old"
print(msg)          # I am 25 years old

# bool conversions
print(bool(0))       # False  (0 is falsy)
print(bool(42))      # True   (non-zero is truthy)
print(bool(""))      # False  (empty string is falsy)
If you try int("hello") you'll get a ValueError! Only convert strings that actually contain a number.

Lesson 4

Naming Conventions & Multiple Assignment

Python has widely adopted naming conventions that make code more readable. Plus, Python lets you assign multiple variables in one line.

Python Naming Conventions

🐍

snake_case for variables & functions

Use lowercase letters with underscores between words.

first_name · total_score · is_valid

🏛️

UPPER_CASE for constants

Values that never change are written in all caps.

MAX_RETRIES = 3 · PI = 3.14159 · API_URL = "…"

🧱

PascalCase for classes

Each word starts with a capital letter (you'll use this in Module 10).

StudentRecord · BankAccount · HttpRequest

🔒

_single_underscore prefix = "private"

Convention for internal-use variables (not enforced by Python).

_internal_count · _helper_data

Multiple Assignment

// Same value
a = b = c = 0
a=0, b=0, c=0
// Tuple unpack
x, y, z = 1, 2, 3
x=1, y=2, z=3
// Swap values
a, b = b, a
values swapped!
// Augmented
score += 10  # score = score + 10
shorthand!
assignment.py
# Multiple assignment on one line
x, y, z = 10, 20, 30
print(x, y, z)      # 10 20 30

# Swap without a temp variable (Python magic!)
a, b = 5, 99
print(a, b)          # 5 99
a, b = b, a
print(a, b)          # 99 5

# Augmented assignment operators
score = 50
score += 10          # score = score + 10
score *= 2           # score = score * 2
print(score)         # 120
Lesson 5

f-Strings — Embedding Variables in Text

f-strings (formatted string literals) are the modern, elegant way to embed variable values directly inside strings. Just prefix your string with f.

📊 f-String in Action
name = "Alice"   age = 25
print(f"Hello, {name}!")
→
Hello, Alice!
print(f"Age: {age}, next year: {age + 1}")
→
Age: 25, next year: 26
print(f"PI ≈ {3.14159:.2f}")
→
PI ≈ 3.14
fstrings.py
name  = "Vinodhini"
role  = "Data Scientist"
years = 5
score = 98.567

# Basic f-string
print(f"Name: {name}")

# Expression inside {}
print(f"{name} has {years * 12} months of experience")

# Format decimals with :.2f
print(f"Score: {score:.2f}")

# String methods inside {}
print(f"Role: {role.upper()}")

# Padding / alignment
print(f"{'Item':10} {'Price':>8}")
print(f"{'Coffee':10} {'3.50':>8}")
f-strings were introduced in Python 3.6 and are now the recommended way to format strings. They're faster and cleaner than the older % and .format() methods.

Old Way (avoid) ❌

"Hello " + name + "!"
"Age: %d" % age
"Hello {}".format(name)

f-String Way ✅

f"Hello {name}!"
f"Age: {age}"
f"{score:.2f}"

Exercise 01 Student Profile

Create four variables: student_name (str), student_age (int), gpa (float), and is_enrolled (bool). Then use an f-string to print: "Alice, 20 yrs old | GPA: 3.80 | Enrolled: True"

Use :.2f inside your f-string to format the GPA to 2 decimal places.
Python
student_name = "Alice"
student_age  = 20
gpa          = 3.8
is_enrolled  = True

print(f"{student_name}, {student_age} yrs old | GPA: {gpa:.2f} | Enrolled: {is_enrolled}")

Output: Alice, 20 yrs old | GPA: 3.80 | Enrolled: True

Exercise 02 Type Detective

For each of the following values, predict the type before running. Then confirm using type():   "99"   99   99.0   True   False + 1

Python
print(type("99"))        # <class 'str'>
print(type(99))          # <class 'int'>
print(type(99.0))        # <class 'float'>
print(type(True))         # <class 'bool'>
print(type(False + 1))    # <class 'int'> — bool + int = int!
Exercise 03 Swap & Augment

Start with a = 10 and b = 25. First swap them using Python's one-line swap. Then double a using augmented assignment. Print the final values of both.

Python
a = 10
b = 25

# Step 1: swap
a, b = b, a
print(f"After swap:  a={a}, b={b}")   # a=25, b=10

# Step 2: double a
a *= 2
print(f"After double: a={a}, b={b}")  # a=50, b=10
