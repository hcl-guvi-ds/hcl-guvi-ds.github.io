📘 Lesson 2
Basic Data Types
Every value in Python has a type. Here are the four you'll use most.
🔢

int - Integer

Whole numbers, positive or negative. No decimal point.

42 -7 0 1_000_000
x = 42
print(x + 10) # 52
print(type(x)) # <class 'int'>
💧

float - Floating Point

Numbers with decimal points. Used for measurements.

3.14 -0.5 2.0 1.5e10
pi = 3.14159
print(pi * 2) # 6.28318
print(round(pi, 2)) # 3.14
📝

str - String

Text data. Any characters inside quotes.

"Hello" 'World' "42" ""
greeting = "Hello, Python!"
print(greeting.upper()) # HELLO, PYTHON!
print(greeting[0]) # H
✅

bool - Boolean

Only two values: True or False. Foundation of logic.

True False 5 > 3 10 == 20
is_raining = False
is_sunny = True
print(is_sunny and not is_raining) # True
📘 Lesson 3
Type Conversion (Casting)
Convert values from one type to another using built-in functions.
conversion.py
s = "100"
n = int(s) # str → int
print(n + 5) # 105

x = float(7) # int → float
print(x) # 7.0

age = 25
msg = "I am " + str(age) + " years old"
print(msg) # I am 25 years old
105
7.0
I am 25 years old
⚠️
Converting int("hello") will raise a ValueError! Only convert strings that actually contain numbers.

📘 Lesson 4
Naming Conventions & Multiple Assignment
Write readable, Pythonic code with proper naming and clever assignment tricks.
🐍 Snake Case

Use lowercase with underscores: first_name, total_score, is_valid.

🏛️ Constants

UPPER_CASE for values that never change: MAX_RETRIES = 3, PI = 3.14159.

assignment.py
# Multiple assignment
x, y, z = 10, 20, 30
print(x, y, z) # 10 20 30

# Swap without a temp variable (Python magic!)
a, b = 5, 99
a, b = b, a
print(a, b) # 99 5

# Augmented assignment
score = 50
score += 10
score *= 2
print(score) # 120
10 20 30
99 5

📘 Lesson 5
f-Strings
The modern, elegant way to embed variables directly in strings.
fstrings.py
name = "Alice"
age = 25
score = 98.567

print(f"Hello, {name}!") # Hello, Alice!
print(f"Next year: {age + 1}") # Next year: 26
print(f"Score: {score:.2f}") # Score: 98.57
Hello, Alice!
Next year: 26
Score: 98.57

Old Way (avoid) ❌

"Hello " + name + "!"
"Age: %d" % age

f-String Way ✅

f"Hello {name}!"
f"Age: {age}"
