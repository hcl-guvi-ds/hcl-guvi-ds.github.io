📘 Lesson 1
What are Conditionals?
Every day we make decisions. Python does the same — it checks conditions and chooses what to do.

🌧️ Real-World Decision

If it is raining → take umbrella.
Else → leave umbrella at home.

🐍 Python Equivalent

if raining:
    take_umbrella()
else:
    leave_umbrella()

Is the number even?

Conditionals use Boolean expressions — things that are either True or False, like 5 > 3 (True) or 10 == 20 (False).
📘 Lesson 2
The if Statement
Runs a block of code only if the condition is True.
if_demo.py
age = 20
if age >= 18:
    print("You are an adult ✅")
print("Program continues...")
You are an adult ✅
Program continues...
⚠️
Indentation is NOT optional! Python uses 4 spaces to know what's inside the if block. Missing indent = IndentationError.
💡
If the condition is False, Python simply skips the indented block and continues with the rest of the program.
📘 Lesson 3
if / else
Two paths — one for True, one for False. One always runs.
if_else.py
temp = 28
if temp > 30:
    print("It's hot! 🌞 Wear light clothes.")
else:
    print("It's pleasant! 🌤️ Enjoy the weather.")
It's pleasant! 🌤️ Enjoy the weather.

🎮 Try it yourself — Change the temperature

Temperature (°C):
🌤️ It's pleasant! 🌤️ Enjoy the weather.

✅ condition is True

The if block runs.
The else block is skipped.

❌ condition is False

The if block is skipped.
The else block runs.

📘 Lesson 4
if / elif / else
Check multiple conditions in order — Python runs the first True one.
grade_checker.py
score = 75
if score >= 90: grade = "A 🏆"
elif score >= 80: grade = "B ⭐"
elif score >= 70: grade = "C 👍"
elif score >= 60: grade = "D 📖"
else: grade = "F 📋"
print(f"Grade: {grade}")
Grade: C 👍
Enter your score (0-100):
Score 75 → Grade: C 👍
💡
Python stops at the first True condition. Order matters! Put the most specific conditions first, then more general ones.
📘 Lesson 5
Nested Conditionals & Ternary
Put if statements inside other if statements, or use the one-line ternary operator.
OUTER if (age >= 18)
if age >= 18:
INNER if (has_id)
if has_id:
    print("Entry granted ✅")
else:
    print("No ID — denied ❌")
else:
    print("Under 18 — not allowed")
Age:
Has ID:
✅ Entry granted! ✅
ternary.py
age = 20
# One-line ternary operator
status = "adult" if age >= 18 else "minor"
print(status) # adult
print("Pass" if 75 >= 60 else "Fail") # Pass
adult
Pass
⚡
Ternary operators are great for simple assignments, but for complex logic, stick with traditional if/else for readability!

⭐ Exercise 2
Traffic Light

light = "green"
if light == "red": print("Stop! 🔴")
elif light == "yellow": print("Slow down! 🟡")
elif light == "green": print("Go! 🟢")
else: print("Unknown signal")
