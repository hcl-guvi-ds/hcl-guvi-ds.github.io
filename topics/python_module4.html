Lesson 2

The if Statement

The most basic conditional — runs a block of code only if the condition is True. If False, Python simply skips over it.

if_basic.py
age = 20

if age >= 18:
    print("You are an adult ✅")

# This always runs — outside the if block
print("Program continues...")

Lesson 3

if / else

Add an else to handle what happens when the condition is False. One path always runs — never both, never neither.

if_else.py
temp = 28

if temp > 30:
    print("It's hot! 🌞 Wear light clothes.")
else:
    print("It's pleasant! 🌤️ Enjoy the weather.")

Lesson 4

if / elif / else

Exercise 01Voting Eligibility

Write a program that checks if a person can vote. They must be 18 or older. Test with age = 17, then age = 21.

Use an if/else. The condition is age >= 18.
Python
age = 17
if age >= 18:
    print("You can vote! 🗳️")
else:
    print("Too young to vote.")
# Output: Too young to vote.
Exercise 02Traffic Light

Write a program with if/elif/else. Given light = "green", print the correct driving instruction for red, yellow, and green lights.

Python
light = "green"
if light == "red":
    print("🔴 Stop!")
elif light == "yellow":
    print("🟡 Slow down!")
elif light == "green":
    print("🟢 Go!")
else:
    print("❓ Unknown signal")
# Output: 🟢 Go!
Exercise 03BMI Category

Given bmi = 23.5, classify it: <18.5 = Underweight, 18.5–24.9 = Normal, 25–29.9 = Overweight, 30+ = Obese. Print with an f-string.

Python
bmi = 23.5
if bmi < 18.5:
    category = "Underweight"
elif bmi < 25:
    category = "Normal weight"
elif bmi < 30:
    category = "Overweight"
else:
    category = "Obese"
print(f"BMI {bmi} → {category}")
# Output: BMI 23.5 → Normal weight
