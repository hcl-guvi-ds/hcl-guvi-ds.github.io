🏷️ Classification Algorithms

Predict categories and class labels — choose an algorithm to learn

📊

Logistic Regression

Despite the name, it's a classifier! Predicts probabilities and classifies into binary categories.

Beginner 📍

K-Nearest Neighbors (KNN)

Classifies a point based on the majority vote of its K closest neighbors in the dataset.

Beginner 🎲

Naive Bayes

Probabilistic classifier based on Bayes' theorem. Fast and great for text classification.

Beginner ✂️

Support Vector Machine (SVM)

Finds the optimal hyperplane that best separates classes with maximum margin.

Intermediate 🌿

Decision Tree

Splits data using if-else rules at each node. Easy to interpret and visualize.

Intermediate 🌲

Random Forest

Builds many decision trees and takes a majority vote. Robust and highly accurate.

Intermediate ⚡

XGBoost

Extreme Gradient Boosting — one of the most powerful algorithms for structured data.

Advanced 🚀

Gradient Boosting

Builds trees sequentially where each tree corrects the errors of the previous one.
