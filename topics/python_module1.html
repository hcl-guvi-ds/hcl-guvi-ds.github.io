Lesson 1

What is Python?

Python is a high-level, general-purpose programming language designed to be easy to read and write. Its syntax looks almost like plain English.

🕐 A Brief History

1989
Guido van Rossum begins designing Python as a hobby project
1991
Python 0.9.0 is published — the very first public version
2008
Python 3.0 released — the modern version used today
Today
World's #1 most popular language, powering AI, web & science
Python was named after Monty Python's Flying Circus — a British comedy show, not the snake! 🎭

Lesson 2

How Python Works

When you run a Python program, your code travels through a simple pipeline before you see the output on screen.

📝

Your Code

You write Python in a .py file or the terminal

→
⚙️

Interpreter

Python reads & runs it line by line

→
🖥️

Output

Results appear on your screen instantly

Unlike C or Java, Python doesn't need a separate compile step. Write → Run. That's it.

Watch It Run

terminal — python3
$ python3 hello.py
Hello, World! 🐍

✅ Python is Interpreted

Runs line by line. Errors show exactly where they are. Great for learning!

✅ Python is High-Level

You don't manage memory or hardware. Python handles the complex stuff.

Lesson 3

Installing Python

Get Python running on your machine in under 5 minutes. Choose your operating system below.

1

Download the Installer

Visit python.org/downloads and click the big yellow "Download Python 3.x.x" button.

2

☑️ Check "Add to PATH"

At the bottom of the installer screen, tick "Add Python to PATH" before clicking Install Now. This step is critical!

3

Verify in Command Prompt

Open Command Prompt (Win+R → type cmd) and run:

python --version
# Should print: Python 3.x.x
1

Check if Python exists

Open Terminal (Cmd+Space → "Terminal") and type python3 --version. macOS often has Python pre-installed.

2

Install via Homebrew (if needed)

brew install python3

Or download the installer from python.org directly.

1

Ubuntu / Debian

sudo apt update && sudo apt install python3
2

Fedora / RHEL

sudo dnf install python3

Lesson 4

Your First Program: Hello, World!

The universal tradition in programming — every beginner's first program. In Python, it's delightfully simple.

hello.py
# My very first Python program
print("Hello, World!")
print("Python is amazing! 🐍")
print(2024, "will be your Python year!")

Line by Line

L1
# My very first… — This is a comment. Python ignores it completely.
L2
print("Hello, World!") — Calls the print function to show text on screen.
L3
You can print any text in quotes — including emoji!
L4
Pass multiple values separated by commas — Python adds a space between them.

Lesson 5

Comments in Python

Comments are lines that Python completely ignores. They're notes for humans — you, your teammates, or future-you — explaining what the code does.

# This is a comment
→
🙈 Python ignores this line
print("Hello")
→
▶ This line RUNS → outputs: Hello
print("World") # inline
→
▶ Runs, then comment ignored
# print("skipped")
→
🚫 Commented out — won't run
comments.py
# ─────────────────────────────
# Program: Grade Checker
# Author:  Vinodhini
# ─────────────────────────────

score = 87  # student's exam score out of 100

# Print the score
print("Score:", score)
Write comments before complex code to plan what you're going to do. This is called "comment-driven development."

✅ Good Comments

# Calculate compound interest

# Max retries before timeout

❌ Useless Comments

# print hello ← obvious

# x = x + 1 ← just restates the code

Hands-On Exercises

Try each exercise yourself before revealing the solution. Struggling a little is part of learning!

Exercise 01Print Your Introduction

Write a Python program that prints 3 lines: your name, your age, and one thing you hope to build with Python.

Each print() call automatically outputs on a new line.
Python
# My introduction
print("Name: Alice")
print("Age: 20")
print("I want to build: A personal AI assistant!")

Replace the values with your own information!

Exercise 02Python as a Calculator

Use print() to compute and display: 125 + 375, 1000 - 237, and 12 × 15.

Python
print(125 + 375)   # 500
print(1000 - 237)  # 763
print(12 * 15)     # 180

Python evaluates the math and prints just the result — no calculator needed!

Exercise 03Comment Challenge

Predict which lines print and which are skipped. Then try uncommenting line 2 — what changes?

Python
print("Line 1 runs")
# print("Line 2 is commented out")
print("Line 3 runs")

Lines 1 and 3 print. Line 2 is commented out so Python skips it. Remove the # from Line 2 and all three lines will print!
