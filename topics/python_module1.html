📘 Lesson 5
Comments in Python
Comments are lines that Python completely ignores — they're notes for humans.
📝
🌍 Real-World Analogy
Comments are like sticky notes on a recipe — they explain what each step does, but don't affect the actual cooking!
comments.py
# This is a comment — Python skips this line
print("This runs!") # Inline comment after code
# print("This is commented out — won't run")
✅ Good Comments

# Calculate compound interest
# Max retries before timeout

❌ Useless Comments

# print hello ← obvious
# x = x + 1 ← restates the code

Comment Challenge
