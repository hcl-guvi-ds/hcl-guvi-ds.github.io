🏗️ Project 1
Build a Python Calculator
A command-line calculator that handles +, -, ×, ÷ operations with input validation.
🏗️
Think of it like a vending machine
The machine (main loop) keeps running. Each button (function) does one specific job. If you press an invalid button, the machine says "Invalid choice" rather than crashing — that's how we design our calculator!
calculator.py
def add(a, b): return a + b
def subtract(a, b): return a - b
def multiply(a, b): return a * b
def divide(a, b):
    if b == 0: raise ValueError("Cannot divide by zero!")
    return a / b

def main():
    ops = {'1': ('Add', add), '2': ('Subtract', subtract), '3': ('Multiply', multiply), '4': ('Divide', divide)}
    while True:
        print("\n=== Calculator ===")
        for k, (n, _) in ops.items(): print(f" {k}. {n}")
        print(" 5. Exit")
        choice = input("Choose: ")
        if choice == '5': break
        if choice not in ops: print("Invalid choice!"); continue
        try:
            a = float(input("First number: "))
            b = float(input("Second number: "))
            _, func = ops[choice]
            print(f"Result: {func(a, b)}")
        except ValueError as e: print(f"Error: {e}")

main()

import json

def save_tasks(tasks):
    with open("tasks.json", "w") as f: json.dump(tasks, f, indent=2)

def add_task(tasks, name):
    tasks.append({"name": name, "done": False})

def view_tasks(tasks):
    for i, t in enumerate(tasks, 1):
        status = "✅" if t["done"] else "⬜"
        print(f" {i}. {status} {t['name']}")
def complete_task(tasks, idx):
    tasks[idx-1]["done"] = True

Build a Python Quiz Game
Use dictionaries, loops, and score tracking to create an engaging quiz engine.
🎮
Think of it like a game show host
The host reads each question card, shows options, waits for your answer, checks if it's right, and keeps score. Our Python code IS the game show host!
quiz.py
questions = [
    {"q": "What keyword creates a function?", "options": ["func", "def", "fn", "define"], "answer": "def"},
    {"q": "Which data type is mutable?", "options": ["tuple", "string", "list", "int"], "answer": "list"}
]

def run_quiz(questions):
    score = 0
    for i, q in enumerate(questions, 1):
        print(f"\nQ{i}: {q['q']}")
        for j, opt in enumerate(q['options'], 1): print(f" {j}. {opt}")
        ans = input("Your answer: ")
        if ans == q["answer"]:
            print("✅ Correct!"); score += 1
        else: print(f"❌ Wrong! Answer: {q['answer']}")
    pct = score / len(questions) * 100
    print(f"\n🎯 Score: {score}/{len(questions)} ({pct:.0f}%)")

🚀 Advanced Project
Build a Python File Manager
Use the os module and OOP to create a powerful file management tool.
🗃️
Think of it like macOS Finder or Windows Explorer
Those GUI apps interact with your file system. We're building the same functionality in Python code, using the os module — Python's bridge to your operating system.
file_manager.py
import os, glob
class FileManager:
    def __init__(self, path="."):
        self.path = os.path.abspath(path)
    def list_files(self):
        for item in sorted(os.listdir(self.path)):
            full = os.path.join(self.path, item)
            kind = "📁" if os.path.isdir(full) else "📄"
            print(f" {kind} {item}")
    def create_file(self, name, content=""):
        filepath = os.path.join(self.path, name)
        with open(filepath, 'w') as f: f.write(content)
    def search_files(self, pattern):
        matches = glob.glob(os.path.join(self.path, f"*{pattern}*"))
        return [os.path.basename(m) for m in matches]
