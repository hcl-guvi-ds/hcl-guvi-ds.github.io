What is K-Nearest Neighbours? 📌
KNN is a simple, non-parametric algorithm that classifies a new data point based on the majority class of its K closest neighbours in the training data.
🏘️
Real-Life Analogy

Moving to a new neighbourhood: When you move to a new city, you tend to adopt habits similar to your nearest neighbours. If 4 out of 5 closest neighbours prefer coffee, you're likely a coffee person too. KNN works exactly this way — it judges by your company!

💻 KNN from Scratch

Python import numpy as np from collections import Counter def knn_classify(X_train, y_train, x_new, k=3): # Calculate Euclidean distances distances = [ np.sqrt(np.sum((x - x_new)**2)) for x in X_train ] # Get k nearest indices k_idx = np.argsort(distances)[:k] k_labels = [y_train[i] for i in k_idx] # Majority vote return Counter(k_labels).most_common(1)[0][0]
>>> X_train = [[1,2],[3,1],[5,3],[6,5],[4,4]] >>> y_train = ['A','A','B','B','A'] >>> knn_classify(X_train, y_train, [3,3], k=3) → Prediction: 'A' Nearest neighbours: A(×2), B(×1) Majority vote wins! ✓

🧩 KNN — Step by Step

1
Load training data
Store all (feature, label) pairs. KNN is a lazy learner — it memorises everything during training and does all computation at prediction time.
2
Receive new data point
A new, unlabelled point arrives. We need to classify it into one of the known categories using the training data.
3
Calculate all distances
Compute distance from the new point to every single training example. Common metrics: Euclidean, Manhattan, Minkowski.
4
Sort & pick K nearest
Sort all distances (ascending) and select the K smallest. These are the "neighbours" that will vote on the prediction.
5
Majority vote → predict
Count the labels of the K neighbours. The most common label wins! For regression, take the average of their values.
Concept 2 of 3
Distance Metrics 📐
The heart of KNN is measuring "closeness". Different distance metrics can dramatically change which neighbours are chosen — and thus the prediction.
🗺️
Real-Life Analogy

Navigating a city: Euclidean distance is like a bird flying straight-line to destination. Manhattan distance is a taxi driving through city blocks. Both measure "closeness" — but differently. Your choice of route changes who your "nearest neighbours" are!

🔍 When to Use Which Metric — Click to Reveal

📏 Euclidean — Best for?
Continuous, real-valued features in low-dimensional spaces. Works well for geometry, physical measurements, and image data. Sensitive to scale — normalise first!
👆 Click to reveal
🏙️ Manhattan — Best for?
High-dimensional data or when features represent independent categories. Less sensitive to outliers than Euclidean. Good for text and sparse data.
👆 Click to reveal
🎯 Minkowski — What is it?
The generalised form! When p=1 → Manhattan. When p=2 → Euclidean. Adjusting p lets you tune the sensitivity to large differences between features.
👆 Click to reveal
⚠️ The Curse of Dimensionality
In very high dimensions, all points become equidistant. KNN breaks down! Solution: use PCA or feature selection to reduce dimensions before applying KNN.

Concept 3 of 3
Choosing the Right K 🎯
K is the most critical hyperparameter in KNN. Too small → overfitting. Too large → underfitting. Finding the sweet spot is an art backed by mathematics.
🗳️
Real-Life Analogy

Asking friends for advice: If you ask only 1 friend (K=1), their quirky opinion dominates. If you ask 100 friends (K=100), everyone's opinion averages out and nuance is lost. Asking 5–10 trusted friends (optimal K) gives a balanced, reliable decision!

💻 Finding Best K with Cross-Validation

Python from sklearn.neighbors import KNeighborsClassifier from sklearn.model_selection import cross_val_score import numpy as np k_range = range(1, 21, 2) # odd values only scores = {} for k in k_range: knn = KNeighborsClassifier(n_neighbors=k) cv_scores = cross_val_score( knn, X_train, y_train, cv=5 ) scores[k] = cv_scores.mean() best_k = max(scores, key=scores.get) print(f"Best K: {best_k}") print(f"Best CV Score: {scores[best_k]:.4f}")
>>> K=1: 0.9417 K=3: 0.9583 >>> K=5: 0.9667 K=7: 0.9750 ← best >>> K=9: 0.9583 K=11: 0.9500 Best K: 7 Best CV Score: 0.9750 ✓ Use GridSearchCV for exhaustive tuning

📋 K Selection Rules of Thumb

📐 Use √N as starting point
For N training samples, start with K ≈ √N as a baseline guess.
🔢 Always use odd K (binary)
Avoids ties in 2-class problems. K=5 is safer than K=4.
🔄 Cross-validate every time
5-fold or 10-fold CV reliably identifies the optimal K for your data.
⚖️ Balance bias & variance
Plot train/test accuracy vs K. Pick K where test accuracy peaks.
Advanced Concept
Weighted KNN & Evaluation ⚡
Standard KNN treats all K neighbours equally. Weighted KNN gives closer neighbours more influence. Combined with proper evaluation, this makes KNN much more powerful.

⚖️ Weighted vs Standard KNN

🏋️
Analogy

Your closest friend's opinion counts more than a distant acquaintance. Weighted KNN applies the same logic — closer = more vote weight!

Python # Standard KNN — equal votes knn_std = KNeighborsClassifier( n_neighbors=5, weights='uniform' # default ) # Weighted KNN — closer = more weight knn_w = KNeighborsClassifier( n_neighbors=5, weights='distance' # 1/distance ) # Custom weight function def my_weights(dists): return np.exp(-dists**2) # Gaussian knn_gaus = KNeighborsClassifier( n_neighbors=5, weights=my_weights )
>>> Standard KNN accuracy: 0.9333 >>> Distance KNN accuracy: 0.9667 >>> Gaussian KNN accuracy: 0.9667 Weighted KNN handles boundary cases better! Especially useful near decision boundaries.

Implement KNN Classifier from Scratch
NumPy

Write a complete KNN classifier class with fit() and predict() methods using only NumPy. Include Euclidean distance and majority vote.

import numpy as np from collections import Counter class KNNClassifier: def __init__(self, k=3): self.k = k def fit(self, X, y): self.X_train = np.array(X) self.y_train = np.array(y) return self def _euclidean(self, a, b): return np.sqrt(np.sum((a - b)**2)) def predict(self, X): return [self._predict_one(x) for x in X] def _predict_one(self, x): dists = [self._euclidean(x, xt) for xt in self.X_train] k_idx = np.argsort(dists)[:self.k] votes = Counter(self.y_train[k_idx]) return votes.most_common(1)[0][0] # Test it! knn = KNNClassifier(k=3) knn.fit([[1,2],[3,1],[5,4]], ['A','A','B']) print(knn.predict([[2,2]])) # ['A']

Find Optimal K using Elbow Method
scikit-learn

Using the Breast Cancer dataset, iterate over K=1 to 20, compute cross-validation accuracy for each K, and identify the optimal K using the elbow method.

from sklearn.datasets import load_breast_cancer from sklearn.neighbors import KNeighborsClassifier from sklearn.model_selection import cross_val_score from sklearn.preprocessing import StandardScaler import numpy as np X, y = load_breast_cancer(return_X_y=True) scaler = StandardScaler() X_scaled = scaler.fit_transform(X) cv_scores = [] k_range = range(1, 21) for k in k_range: knn = KNeighborsClassifier(n_neighbors=k) scores = cross_val_score( knn, X_scaled, y, cv=5, scoring='accuracy' ) cv_scores.append(scores.mean()) best_k = np.argmax(cv_scores) + 1 print(f"Optimal K: {best_k}") print(f"Best Accuracy: {max(cv_scores):.4f}")

KNN Regression for House Price Prediction

Implement KNN regression to predict house prices. Use features: size (sqft), bedrooms, age. Evaluate using RMSE and compare K=3 vs K=7.

from sklearn.neighbors import KNeighborsRegressor from sklearn.preprocessing import StandardScaler from sklearn.metrics import mean_squared_error import numpy as np # Simulated house data X = [[1200,2,10],[1800,3,5],[900,1,20], [2200,4,3],[1500,3,8],[2800,5,1]] y = [250000,380000,180000, 490000,320000,650000] scaler = StandardScaler() X_scaled = scaler.fit_transform(X) for k in [3, 7]: knn = KNeighborsRegressor(n_neighbors=k) knn.fit(X_scaled, y) pred = knn.predict(X_scaled) rmse = np.sqrt(mean_squared_error(y, pred)) print(f"K={k} → RMSE: ${rmse:,.0f}")
