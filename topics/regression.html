📈 Regression Algorithms

Predict continuous numerical values — choose an algorithm to learn

📏

Linear Regression

The foundation of regression. Fits a straight line through data points to predict a continuous output.

Beginner 📐

Multiple Linear Regression

Extends linear regression to use multiple input features to make better predictions.

Beginner 〰️

Polynomial Regression

Fits a curved line to data. Great for capturing non-linear relationships between variables.

Intermediate 🔵

Ridge Regression (L2)

Adds a penalty to large coefficients to prevent overfitting. Best when all features matter.

Intermediate 🔶

Lasso Regression (L1)

Similar to Ridge but can shrink some coefficients to zero — great for feature selection.

Intermediate 🌿

Decision Tree Regression

Splits data into branches based on feature values to predict a number at each leaf node.

Intermediate 🌲

Random Forest Regression

Builds hundreds of decision trees and averages their predictions for better accuracy.

Intermediate 📡

Support Vector Regression

Uses SVM principles to find the best-fit line within a margin of tolerance.

Advanced
