Lesson 1

Arithmetic Operators

Arithmetic operators perform mathematical calculations on numbers. Python supports all the standard operations plus two extra powerful ones.

OperatorNameExampleResultNotes
+Addition10 + 313Also joins strings
-Subtraction10 - 37
*Multiplication10 * 330Also repeats strings
/Division10 / 33.333…Always returns float
//Floor Division10 // 33Divides & rounds down
%Modulus10 % 31Returns the remainder
**Exponentiation2 ** 82562 to the power of 8
arithmetic.py
a, b = 17, 5

print(a + b)   # 22  — addition
print(a - b)   # 12  — subtraction
print(a * b)   # 85  — multiplication
print(a / b)   # 3.4 — division (always float)
print(a // b)  # 3   — floor division
print(a % b)   # 2   — modulus (remainder)
print(a ** b)  # 1419857 — 17 to the power of 5
The modulus operator % is super useful! Use it to check if a number is even (n % 2 == 0), or to keep numbers in a range.

Comparison Operators

Comparison operators compare two values and always return a Boolean — either True or False. They're the backbone of decision-making in Python.

OperatorMeaningExampleResult
==Equal to5 == 5True
!=Not equal to5 != 3True
>Greater than10 > 3True
<Less than3 < 10True
>=Greater than or equal5 >= 5True
<=Less than or equal4 <= 5True

⚖️ Interactive Comparison — change the values and watch results update

a = b =
comparisons.py
x = 10
y = 5

print(x == y)   # False — 10 is not equal to 5
print(x != y)   # True  — they ARE different
print(x > y)    # True  — 10 is greater
print(x < y)    # False — 10 is not less than 5
print(x >= 10)  # True  — 10 equals 10
print(x <= y)   # False

# Compare strings too!
print("apple" == "apple")  # True
print("a" < "b")           # True (alphabetical)
Don't confuse == (comparison) with = (assignment)! x = 5 stores 5. x == 5 checks if x equals 5.
Lesson 3

Logical Operators

Logical operators combine multiple Boolean expressions. Python has three: and, or, and not. They make complex conditions possible.

🤝

and

Returns True only if both sides are True.

True and True → True
True and False → False
🔀

or

Returns True if at least one side is True.

True or False → True
False or False → False
🔄

not

Flips the Boolean — True becomes False and vice versa.

not True → False
not False → True

logical.py
age   = 20
score = 85

# and — both conditions must be True
print(age >= 18 and score >= 80)  # True
print(age >= 18 and score >= 90)  # False (score is only 85)

# or — at least one must be True
print(age >= 18 or score >= 90)   # True (age condition passes)
print(age < 18 or score < 60)    # False (both fail)

# not — flip the result
print(not (age >= 18))            # False
print(not False)                  # True
Think of and as a strict gate (everyone must pass), or as a lenient gate (anyone can pass), and not as a flipper.

precedence.py
# Without parentheses — follows precedence
print(2 + 3 * 4)     # 14  (* before +)
print(2 ** 3 + 1)   # 9   (** before +)
print(10 / 2 + 3)   # 8.0 (/ before +)

# With parentheses — override the order!
print((2 + 3) * 4)  # 20  (+ first now)
print(10 / (2 + 3)) # 2.0 (+ first now)

Exercise 01 Odd or Even Checker

Use the modulus operator to check if the number 47 is odd or even. Print "47 is odd" or "47 is even" using an f-string. Then test with 100.

Hint: A number is even if n % 2 == 0. The result of % is the remainder.
Python
n = 47
if n % 2 == 0:
    print(f"{n} is even")
else:
    print(f"{n} is odd")   # 47 is odd

n = 100
print(f"{n} is {'even' if n % 2 == 0 else 'odd'}")  # 100 is even
Exercise 02 Voter Eligibility Check

A person can vote if they are 18 or older AND a citizen. Create variables age = 20 and is_citizen = True. Use logical operators to print whether the person can vote.

Python
age = 20
is_citizen = True

can_vote = age >= 18 and is_citizen
print(f"Can vote: {can_vote}")        # Can vote: True

# Test with a non-citizen
is_citizen = False
can_vote = age >= 18 and is_citizen
print(f"Can vote: {can_vote}")        # Can vote: False
Exercise 03 Predict the Output

Without running the code, predict the output of each line. Then check by running them in Python.

Python
print(3 + 4 * 2)        # ?
print((3 + 4) * 2)      # ?
print(2 ** 3 ** 2)      # ?  (hint: ** is right-associative)
print(15 % 4)           # ?
print(True and False or True)  # ?
Answers
print(3 + 4 * 2)        # 11  (* first: 4*2=8, then 3+8=11)
print((3 + 4) * 2)      # 14  (parens first: 3+4=7, then 7*2=14)
print(2 ** 3 ** 2)      # 512 (right to left: 3**2=9, then 2**9=512)
print(15 % 4)           # 3   (15 = 3*4 + 3, remainder=3)
print(True and False or True)  # True (and first: False or True = True)
