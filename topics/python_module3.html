Logical Operators
Combine conditions — and, or, not.
🤝

and

True only if both are True

True and True → True
🔀

or

True if at least one is True

True or False → True
🔄

not

Flips the value

not True → False
logical.py
age = 20
score = 85
print(age >= 18 and score >= 80)
print(age >= 18 or score >= 90)
