📘 Concept 1
The for Loop
Iterate over any sequence — lists, strings, ranges, and more.
🍕
🌍 Real-World Analogy
Imagine you have a pizza with 8 slices. A for loop is like saying: "For each slice on this pizza, take a bite." You automatically repeat the action for every item — no need to count manually!
Syntax
for_loop_syntax.py
for item in sequence:
    # code to repeat for each item
    print(item)
example.py
# Loop over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print("I love", fruit)
I love apple
I love banana
I love cherry

📘 Concept 2
The while Loop
Repeat a block of code as long as a condition remains True.
🚦
🌍 Real-World Analogy
A while loop is like a traffic light: "While the light is red, wait." You keep doing something until the condition changes. Once it turns green (False), you stop!
Syntax
while_loop.py
count = 0
while count < 5:
    print("Count is:", count)
    count += 1
print("Done!")
Count is: 0
Count is: 1
Count is: 2
Count is: 3
Count is: 4
Done!

📘 Concept 3
Loop Control Statements
break, continue, and pass — control what happens inside your loop.
🎢
🌍 Real-World Analogy
Think of checking items in a queue:
• break = Emergency stop. Close the queue immediately.
• continue = Skip this person, move to next.
• pass = Do nothing right now, keep going as normal.

📘 Concept 4
Nested Loops
A loop inside another loop — powerful for 2D data, grids, and combinations.
🗂️
🌍 Real-World Analogy
Think of a school timetable: "For each day of the week, for each period in the day, show the subject." That's a nested loop — the outer loop goes through days, the inner loop through periods.
nested_loop.py
for row in range(3):
    for col in range(3):
        print(f"({row},{col})", end=" ")
    print() # new line after each row
(0,0) (0,1) (0,2)
(1,0) (1,1) (1,2)
(2,0) (2,1) (2,2)

⚠️
Performance tip: A 3×3 nested loop runs 9 iterations. A 100×100 runs 10,000! Be mindful of nested loops with large datasets.
⭐ Advanced Concept
List Comprehensions
Create lists in a single, Pythonic line — powered by loops!
⚡
🌍 Real-World Analogy
A list comprehension is like ordering a custom pizza: "Give me all toppings that are vegetarian, doubled." Instead of picking each one, you describe the rule — Python does the rest instantly.
Traditional Loop vs List Comprehension
Traditional (5 lines)
squares = []
for x in range(5):
  squares.append(x ** 2)
print(squares)
List Comprehension (1 line!) ✨
squares = [x**2 for x in range(5)]
print(squares)
[0, 1, 4, 9, 16]

💡 Pro Tips
✅
Use comprehensions for simple transformations — they are faster and more Pythonic.
⚠️
Avoid putting too much logic in one comprehension — readability matters more than cleverness!

Exercise 1: Sum of Numbers
Write a Python program using a for loop that calculates the sum of all numbers from 1 to 10 (inclusive). Print the final sum.

⭐⭐ Medium
Exercise 2: FizzBuzz
Using a for loop with range(1, 21), print "Fizz" for multiples of 3, "Buzz" for multiples of 5, "FizzBuzz" for multiples of both, otherwise print the number.
fizzbuzz.py
for n in range(1, 21):
    if ___:
        print("FizzBuzz")
    elif ___:
        print("Fizz")
    elif ___:
        print("Buzz")
    else:
        print(n)
fizzbuzz_solution.py
for n in range(1, 21):
    if n % 15 == 0:
        print("FizzBuzz")
    elif n % 3 == 0:
        print("Fizz")
    elif n % 5 == 0:
        print("Buzz")
    else:
        print(n)
💡
Check FizzBuzz first (n % 15) to catch multiples of both 3 and 5!
⭐⭐⭐ Hard
Exercise 3: Find First Prime
Use a while loop and break to find the first prime number greater than 20. A prime is divisible only by 1 and itself.
first_prime.py
num = 21
while True:
    is_prime = True
    for i in range(2, num):
        if num % i == 0:
            is_prime = False
            break
    if is_prime:
        print("First prime > 20:", num)
        break
    num += 1
✅
Output: First prime > 20: 23
