Why functions? They let you write code once and reuse it anywhere. This is called the DRY principle: Don't Repeat Yourself!
📘 Concept 1
What is a Function?
A named, reusable block of code that performs a specific task.
☕
Real-Life Analogy: Coffee Machine
A coffee machine is a function. You press a button (call the function), it processes (executes the body), and gives you coffee (returns output). You don't re-wire it every time!

The def keyword defines a function. The code inside runs only when you call the function by name!
📘 Concept 2
Parameters & Arguments
Parameters are the variables in the definition; arguments are the actual values you pass when calling.
📋

Positional

Matched by position

def add(a, b): return a + b
add(3, 5) # a=3, b=5
🏷️

Default Values

Optional with fallbacks

def greet(name, msg="Hello"):
  return msg + " " + name
greet("Alice")
🔑

Keyword Args

Named, any order

def profile(name, age, city): pass
profile(age=25, name="Bob", city="NYC")
📦

*args

Any number

def total(*nums): return sum(nums)
total(1,2,3,4,5) # 15
⚙️ def power(base, exponent=2, label="Result")

parameters.py
def introduce(name, role="Student", city="India"):
    print(f"Hi! I'm {name}, a {role} from {city}.")

introduce("Alice")

⚠️
Default parameters must come after required parameters. def fn(required, optional="default") ✅

📘 Concept 4
Scope & Variables
Scope determines where a variable can be accessed. Local (inside functions) vs Global (anywhere).
🌍 GLOBAL SCOPE
message = "Hello World"
count = 0
🔵 LOCAL SCOPE (inside function)
x = 10
y = 20
✅ Can see: x, y, message
❌ message read-only without global
✅ Can see: message, count
❌ Cannot see: x, y (local)
scope.py
name = "Global Alice"
def show():
    name = "Local Bob"
    print(f"Inside: {name}")
show()
print(f"Outside: {name}")
Inside: Local Bob
Outside: Global Alice
💡
Best practice: Avoid using global variables inside functions. Pass values as parameters and return results — this makes functions predictable and testable.

advanced.py
# Lambda — anonymous function
square = lambda x: x ** 2
print(square(5))

# Recursion — function calls itself
def factorial(n):
    if n <= 1: return 1
    return n * factorial(n - 1)

print(factorial(5))
25
120
⚠️
Every recursive function needs a base case — a condition where it stops calling itself. Without it, you get infinite recursion and a RecursionError!

⭐ Exercise 1
Temperature Converter
Write a function celsius_to_fahrenheit(celsius) that converts Celsius to Fahrenheit using: F = (C × 9/5) + 32. Test with 0, 100, and 37.
def celsius_to_fahrenheit(c):
    return (c * 9/5) + 32
print(celsius_to_fahrenheit(0)) # 32.0
print(celsius_to_fahrenheit(100)) # 212.0
print(celsius_to_fahrenheit(37)) # 98.6
⭐ Exercise 2
Palindrome Checker
Write a function is_palindrome(word) that returns True if the word reads the same forwards and backwards.
def is_palindrome(word):
    word = word.lower()
    return word == word[::-1]
print(is_palindrome("racecar")) # True
print(is_palindrome("hello")) # False
⭐ Exercise 3
Grade Calculator with *args
Write a function average_grade(*scores) that takes any number of scores and returns the average and letter grade (A≥90, B≥80, C≥70, D≥60, F).
def average_grade(*scores):
    avg = sum(scores) / len(scores)
    grade = 'A' if avg >= 90 else 'B' if avg >= 80 else 'C' if avg >= 70 else 'D' if avg >= 60 else 'F'
    return avg, grade
print(average_grade(85, 92, 78, 90))
