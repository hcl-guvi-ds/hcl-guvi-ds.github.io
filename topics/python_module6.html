Concept 1 of 4
Python Lists 📋
A list is an ordered, mutable collection of items. You can store anything — numbers, strings, even other lists!
🛒
Real-Life Analogy

Think of a shopping cart. You can add items, remove them, rearrange them, and the order matters (item 1 is at index 0). That's exactly how a Python list works!

🗂️ Step-by-Step: How append() works

1
Start with existing list
Python keeps a list in memory: fruits = ["apple", "banana"]
2
Call .append()
fruits.append("cherry") — Python finds the end of the list and allocates a new slot.
3
New element inserted at index 2
"cherry" is placed at index 2. List length becomes 3.
4
Result: updated list
["apple", "banana", "cherry"] — in order, mutable, indexed from 0.
Concept 2 of 4
Python Tuples 🔒
A tuple is like a list, but immutable — once created, you cannot change its elements. Perfect for data that should never change.
🗿
Real-Life Analogy

Think of a birth certificate. The data on it (name, date, place) is fixed forever. You can read it, but you can't modify it. That's a tuple!

💻 Code Example

1# Creating a tuple 2coords = (28.7, 77.1) 3 4print(coords[0]) # 28.7 5 6# Tuple unpacking 7lat, lon = coords 8print(lat) # 28.7
>>> 28.7 >>> lat = 28.7, lon = 77.1

Concept 3 of 4
Python Sets 🔵
A set is an unordered collection of unique elements. Duplicates are automatically removed. Perfect for membership testing and mathematical operations.
🎡
Real-Life Analogy

Think of a raffle draw jar. Each ticket number can only appear once. There's no specific order, and you can check if a number is in the jar instantly. That's a Python set!

💻 Code Example

Python 1# Creating a set (duplicates removed!) 2nums = {1, 2, 2, 3, 3, 4} 3print(nums) # {1, 2, 3, 4} 4 5# Set operations 6a = {1, 2, 3} 7b = {3, 4, 5} 8print(a & b) # {3} ← intersection 9print(a | b) # {1,2,3,4,5} ← union
>>> {1, 2, 3, 4} >>> {3} >>> {1, 2, 3, 4, 5}

Concept 4 of 4
Python Dictionaries 📖
A dictionary stores data as key-value pairs. Think of it as a real dictionary — you look up a word (key) to get its definition (value).
📞
Real-Life Analogy

Think of a phone book. Each person's name (key) maps to their phone number (value). You look up by name, not by position. Dictionaries work exactly like this!

💻 Code Example

Python 1# Creating a dictionary 2student = { 3 "name": "Vinodhini", 4 "age": 25, 5 "course": "Data Science" 6} 7 8# Accessing a value by key 9print(student["name"]) # Vinodhini 10 11# Adding a new key 12student["grade"] = "A" 13print(student.keys())
>>> Vinodhini >>> dict_keys(['name', 'age', 'course', 'grade'])

🔁 Dictionary Methods

print(student.keys()) # dict_keys(['name', 'age', 'course', 'grade', 'active'])

Returns all keys in the dictionary.

print(student.values()) # dict_values(['Vinodhini', 25, 'Data Science', 'A', True])

Returns all values in the dictionary.

print(student.items()) # dict_items([('name','Vinodhini'), ('age',25), ...])

Returns all key-value pairs as tuples.

student.get("score", "Not Found") # "Not Found" — safer than [] access

Safely retrieves a value; returns a default if key doesn't exist.

1
List — Remove Duplicates Without Using a Set
List

Given nums = [1, 2, 2, 3, 4, 4, 5], create a new list with no duplicates while preserving order.

nums = [1, 2, 2, 3, 4, 4, 5] seen = [] for n in nums: if n not in seen: seen.append(n) print(seen) # [1, 2, 3, 4, 5]
2
Dictionary — Word Frequency Counter
Dict

Given words = ["apple","banana","apple","cherry","banana","apple"], count how many times each word appears.

words = ["apple","banana","apple","cherry","banana","apple"] freq = {} for w in words: freq[w] = freq.get(w, 0) + 1 print(freq) # {'apple': 3, 'banana': 2, 'cherry': 1}
3
Set — Common Elements Between Two Lists
Set

Given a = [1,2,3,4] and b = [3,4,5,6], find elements that appear in both lists.

a = [1,2,3,4] b = [3,4,5,6] common = set(a) & set(b) print(common) # {3, 4}
