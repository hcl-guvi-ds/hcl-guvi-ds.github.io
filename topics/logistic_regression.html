📊 Classification Basics
Understanding binary classification problems
What is Classification?
Classification is a supervised learning task where we predict categories (classes) rather than continuous values. Binary classification focuses on predicting one of two possible outcomes.
💡 Real-Life Analogy
Email spam detection: You train a model on emails labeled "Spam" or "Not Spam". The model learns patterns to classify new emails. Logistic regression is perfect for this task!

🎯 Real-World Examples

Class 0: Not spam
Class 1: Spam

🏥 Disease Diagnosis

📈 Sigmoid Function & Probabilities
Converting linear outputs to probabilities
The Sigmoid Function
The sigmoid function squashes any input value to a probability between 0 and 1. It's the key to logistic regression: σ(z) = 1 / (1 + e^(-z))
💡 Real-Life Analogy
Like a smooth ramp from "definitely No" to "definitely Yes". Small changes in the middle produce big probability shifts, while extreme values plateau near 0 or 1.
🎮 Adjust Sigmoid Input
0.00
Probability of Class 1:
0.50
Mathematical Expression
# Sigmoid function implementation import numpy as np def sigmoid(z): """Sigmoid function: σ(z) = 1 / (1 + e^(-z))""" return 1 / (1 + np.exp(-z)) # Example values z_values = np.array([-5, -2, 0, 2, 5]) probabilities = sigmoid(z_values) print(probabilities) # [0.0067, 0.1192, 0.5, 0.8808, 0.9933]

🎯 Logistic Regression Model
Combining linear regression with sigmoid function
How Logistic Regression Works
Logistic regression combines a linear model with the sigmoid function. First, it computes a weighted sum of features, then passes it through sigmoid to get a probability between 0 and 1.
💡 Real-Life Analogy
Like a recommendation system: Features (age, preferences) go through a weighted calculation, then a sigmoid gate ensures the output is a probability "Yes/No" rather than unbounded values.
🎮 Model Architecture Flow
Features (X) → Weight Sum
↓
z = w₀ + w₁x₁ + w₂x₂ + ...
↓
Apply Sigmoid σ(z)
↓
Probability P(y=1) ∈ [0, 1]
↓
Classify: If P > 0.5 → Class 1, Else → Class 0
🎯 Decision Threshold Explorer
0.50
Classification Rule: If P(y=1) > 0.50 → Predict Class 1, else → Class 0
Training & Cost Function
Logistic regression uses Log Loss (Cross-Entropy) to measure errors and trains with gradient descent to minimize this loss.
from sklearn.linear_model import LogisticRegression from sklearn.metrics import accuracy_score # Create and train model model = LogisticRegression(random_state=42) model.fit(X_train, y_train) # Make predictions predictions = model.predict(X_test) probabilities = model.predict_proba(X_test) # Evaluate accuracy = accuracy_score(y_test, predictions) print(f'Accuracy: {accuracy:.4f}')

🚀 Advanced Topics
Model evaluation, regularization, and multiclass classification
Model Evaluation Metrics
Beyond accuracy, we use multiple metrics to evaluate classification models comprehensively.
📊 Evaluation Metrics
🎯 Accuracy
Formula: (TP + TN) / Total
Use Case: Balanced datasets
Range: 0 to 1
🎖️ Precision
Formula: TP / (TP + FP)
Use Case: When false positives are costly
Meaning: Of predicted positives, how many are correct?
🔔 Recall (Sensitivity)
Formula: TP / (TP + FN)
Use Case: When false negatives are costly
Meaning: Of actual positives, how many did we find?
📈 ROC-AUC
Meaning: Area Under Receiver Operating Characteristic Curve
Range: 0 to 1 (higher is better)
Use Case: Overall model discrimination ability
Regularization Techniques
Prevent overfitting by adding penalties to the cost function.
🔧 Regularization Types
L2 (Ridge)
L1 (Lasso)
ElasticNet
L2 Regularization (Ridge)
Adds penalty proportional to square of coefficients. Shrinks all weights but doesn't eliminate them.
Complete Implementation with Evaluation
from sklearn.linear_model import LogisticRegression from sklearn.metrics import ( accuracy_score, precision_score, recall_score, roc_auc_score ) # Train with regularization model = LogisticRegression( C=1.0, # Inverse regularization strength penalty='l2' # Ridge regularization ) model.fit(X_train, y_train) # Predictions y_pred = model.predict(X_test) y_proba = model.predict_proba(X_test)[:, 1] # Comprehensive evaluation accuracy = accuracy_score(y_test, y_pred) precision = precision_score(y_test, y_pred) recall = recall_score(y_test, y_pred) auc = roc_auc_score(y_test, y_proba)
