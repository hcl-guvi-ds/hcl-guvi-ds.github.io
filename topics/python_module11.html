Concept 1

What are Libraries?

A library (also called a module or package) is a collection of pre-written Python code that you can reuse in your programs. Instead of writing everything from scratch, you import a library and use its ready-made tools.

🧰

Real-Life Analogy: A Toolbox

Imagine building a house. You don't forge your own hammer or manufacture screws — you open a toolbox full of ready-made tools. Python libraries are your toolbox. You pick the right tool for the job and get building!

os & datetime Libraries

The os module lets you interact with your operating system — files, folders, environment. datetime gives you powerful tools for working with dates and times.

Exercise 01Math Statistics Calculator

Using the math and statistics libraries, write a function analyze(numbers) that takes a list and returns: mean, standard deviation, min, max, and the square root of the mean. Test with [4, 9, 16, 25, 36].

import statistics gives you statistics.mean(), statistics.stdev() etc.
solution.py
import math
import statistics

def analyze(numbers):
    mean   = statistics.mean(numbers)
    std    = statistics.stdev(numbers)
    mn     = min(numbers)
    mx     = max(numbers)
    sqrt_m = math.sqrt(mean)
    print(f"Mean:   {mean:.2f}")
    print(f"StdDev: {std:.2f}")
    print(f"Min:    {mn} | Max: {mx}")
    print(f"√Mean:  {sqrt_m:.4f}")

analyze([4, 9, 16, 25, 36])
Exercise 02Random Password Generator

Using random and string libraries, write a function generate_password(length=12, include_symbols=True) that creates a secure random password. It should include letters, digits, and optionally symbols.

solution.py
import random
import string

def generate_password(length=12, include_symbols=True):
    chars = string.ascii_letters + string.digits
    if include_symbols:
        chars += string.punctuation
    pwd = "".join(random.choice(chars) for _ in range(length))
    return pwd

print(generate_password())           # 12 chars with symbols
print(generate_password(8, False))  # 8 chars, no symbols
print(generate_password(20))         # 20 chars with symbols
Exercise 03File Organizer with os

Write a function count_files_by_extension(folder_path) using the os module that counts how many files of each extension are in a directory and returns a dictionary like {"py": 5, "txt": 3}.

solution.py
import os

def count_files_by_extension(folder_path="."):
    counts = {}
    for filename in os.listdir(folder_path):
        if os.path.isfile(os.path.join(folder_path, filename)):
            # get extension (e.g. ".py" → "py")
            ext = os.path.splitext(filename)[1].lstrip(".")
            ext = ext if ext else "no_ext"
            counts[ext] = counts.get(ext, 0) + 1
    return counts

result = count_files_by_extension()
for ext, count in sorted(result.items()):
    print(f".{ext}: {count} file(s)")
